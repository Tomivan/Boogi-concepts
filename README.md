Boogi Noire Fragrance - An e-commerce website for a perfume business

This is a website where you can come and shop for different fragrances for men and women. This application was built using React, with Bootstrap for styling purposes and Firebase as the backend and database. Paystack was used as the payment gateway.

To install this project, clone the repository and run npm install to install all the packages necessary to use this app effectively. There is a .env file that contains the Firebase configuration and the api key for integrating Paystack.
